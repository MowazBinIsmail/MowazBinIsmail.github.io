# Mowaz Bin Ismail's Personal Website

![Website Preview](images/M1.jpeg)

## 🌟 Overview

This is my personal website showcasing my journey, interests, and projects. The website features a modern, interactive design with glassmorphism effects and smooth animations.

## 🚀 Features

- **Modern Design**: Implements contemporary design patterns including glassmorphism and gradient effects
- **Responsive Layout**: Fully responsive design that works seamlessly across all devices
- **Interactive Elements**: Engaging hover effects and smooth transitions
- **Social Media Integration**: Connected with various social platforms
- **Project Showcase**: Dedicated section for highlighting projects and achievements

## 🛠 Technologies Used

- HTML5
- CSS3 (SCSS)
- JavaScript (jQuery)
- Font Awesome Icons
- Google Fonts

## 📂 Project Structure

```
├── index.html              # Main HTML file
├── assets/
│   ├── css/               # Compiled CSS files
│   ├── js/                # JavaScript files
│   ├── sass/              # SCSS source files
│   └── fonts/             # Font files
└── images/                # Image assets
```
